# Compound Interest Calculator

  A simple Android application that calculates compound interest using interactive sliders
  and visualizes the result with an animated bar chart.

  **Formula used:** `A = P × (1 + r/100)^t`

  ---

  ## 📱 Features

  - Interactive sliders for **deposit amount**, **interest rate**, and **time period**
  - Real-time calculation of total value and interest earned
  - Animated **bar chart** showing the breakdown between principal and interest
  - Localized number formatting for Slovak locale
  - Tooltips displayed above sliders during adjustment

  ---

  ## 🛠️ Tech Stack

  - **Language:** Kotlin
  - **Platform:** Android
  - **Build system:** Gradle (Kotlin DSL)
  - **Chart library:** MPAndroidChart

  ---

  ## 🧮 Default Values

  | Parameter     | Default |
  |---------------|---------|
  | Deposit       | 10,000  |
  | Interest rate | 5%      |
  | Time period   | 5 years |
